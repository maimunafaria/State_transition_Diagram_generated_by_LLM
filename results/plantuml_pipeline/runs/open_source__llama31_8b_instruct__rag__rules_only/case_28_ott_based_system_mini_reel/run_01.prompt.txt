## Hide empty descriptions

Use `hide empty description` to render states as simple boxes when they have no internal text.

```plantuml
@startuml
hide empty description
[*] --> Idle
Idle --> Processing
Processing --> [*]
@enduml
```

## Inline color

Individual states may include inline color.

```plantuml
state CurrentSite #pink {
  [*] --> Loading
  Loading --> Ready
}
```

## Skin parameters

`skinparam` can control drawing style, colors, and fonts.

```plantuml
skinparam state {
  BackgroundColor LightYellow
  BorderColor Black
}
```

## Long state names with alias

Use aliases to keep transitions readable when state names are long.

```plantuml
state "Waiting for Payment Confirmation" as WaitingPayment
state "Order Completed Successfully" as OrderComplete

WaitingPayment --> OrderComplete : payment confirmed
```

Alternative alias form:

```plantuml
state WaitingPayment as "Waiting for Payment Confirmation"
```
