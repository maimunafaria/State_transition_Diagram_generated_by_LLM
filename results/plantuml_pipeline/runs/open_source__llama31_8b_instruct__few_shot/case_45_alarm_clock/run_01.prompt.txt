Example 1 Requirement:
Cd Player — Polished Requirement Specification

Functional Requirements
1. The system shall start in a stopped state.
2. The system shall begin playing music when a disc is inserted and the user presses play.
3. The system shall allow the user to pause the music at any time while it is playing.
4. The system shall stop the music temporarily when paused but allow resuming from the same point.
5. The system shall allow the user to stop the player at any time, regardless of whether the music is playing or paused.
6. The system shall return to a stopped state when the player is stopped.
7. The system shall allow switching to another disc and continuing playback if multiple discs are available.
8. The system shall remain in a stopped state when no more discs are left.

@startuml

Playing --> Paused : press pause
Paused --> Playing : press play
Paused --> Stopped : press stop

Playing --> Playing : [another disc] / change disc

@enduml

--- Process ---

1. Identify all system states.

2. Identify events/conditions triggering transitions.

3. Define transitions clearly between states.

4. Ensure logical consistency (no unreachable states).

5. Identify one initial state and at least one final state.



--- Output Rules ---

- Generate ONLY valid PlantUML code.

- Include initial and final states.

- Use proper UML state diagram syntax.

- Do not include explanations or extra text.

Alarm Clock — Polished Requirement Specification

Functional Requirements
1. The system shall be in an off state initially.
2. The system shall support when the user sets an alarm time, the system shall activate the alarm.
3. The system shall ensure that the user shall be able to turn off the activated alarm.
4. The system shall support when the current time matches the set alarm time, the system shall start ringing the alarm.
5. The system shall ensure that once the alarm is ringing, the user shall be able to stop it.
6. The system shall support after stopping the alarm, the system shall remain active for the next scheduled alarm time.
