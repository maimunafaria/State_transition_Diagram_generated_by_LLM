Atm Scenario 1 — Polished Requirement Specification

Functional Requirements
1. The system shall allow users to choose or enter a withdrawal amount.
2. The system shall support predefined amounts for selection by the user.
3. The system shall allow users to input their own custom withdrawal amount.
4. The system shall provide an option for the user to cancel the transaction at any point.
5. The system shall support if the user cancels the transaction, the system shall stop the transaction and return the card.
