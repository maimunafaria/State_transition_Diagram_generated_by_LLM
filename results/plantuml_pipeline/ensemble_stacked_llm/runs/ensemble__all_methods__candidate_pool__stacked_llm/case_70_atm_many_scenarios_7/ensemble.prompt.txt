Requirement:
Atm Many Scenarios Scenario 7 — Polished Requirement Specification

Functional Requirements
1. The system shall greet the user when they arrive.
2. The system shall check the PIN after the card is inserted.
3. The system shall allow the user to try again if the entered PIN is incorrect.
4. The system shall keep the card for security if it is invalid or the PIN is entered incorrectly too many times.
5. The system shall return the card if the user cancels the operation.
6. The system shall display available options (withdrawing money, checking information, or making a transfer) for the user to choose from if the PIN is correct.
7. The system shall process the selected request after the user makes a choice.
