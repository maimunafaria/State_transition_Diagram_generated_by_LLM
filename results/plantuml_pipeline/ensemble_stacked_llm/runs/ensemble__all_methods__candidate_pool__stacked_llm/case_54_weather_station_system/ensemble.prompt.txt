Requirement:
Weather Station System — Polished Requirement Specification

Functional Requirements
1. The system shall restart and begin running from a shut down condition.
2. The system shall enter configuration mode when changes or power-saving settings are required.
3. The system shall return to running state after configuration is completed.
4. The system shall collect data when triggered by the clock while running.
5. The system shall prepare and send out a weather summary upon request.
6. The system shall check its own status and transmit the result upon request.
7. The system shall return to running state after performing an activity (data collection, summary preparation, or status checking).
8. The system shall move into a controlled mode through remote control.
9. The system shall return to shut down condition if a shutdown command is given while running.
