Requirement:
Farm Management System — Polished Requirement Specification

Functional Requirements
1. The system shall allow a user to log in.
2. The system shall display the main area after a user signs in, allowing them to choose what to check.
3. The system shall show market conditions when the user selects this option.
4. The system shall show the market report if the customer selects market condition.
4. The system shall require users to provide a photo when they choose pest control help.
4. The system shall show the pest control techniques if the customer selects pest control help.
5. The system shall provide soil-related suggestions ( which crops are suitable ) after receiving soil information from the user.
6. The system shall offer weather insights based on the provided location.
7. The system shall display labour availability details after receiving a location from the user.
8. The system shall allow users to continue using the service after viewing any of these results.
9. The system shall end the session when a user logs out.
