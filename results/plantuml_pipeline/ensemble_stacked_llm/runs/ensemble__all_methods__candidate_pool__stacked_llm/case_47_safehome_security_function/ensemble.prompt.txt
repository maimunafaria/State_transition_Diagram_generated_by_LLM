Requirement:
Safehome Security Function — Polished Requirement Specification

Functional Requirements
1. The system shall check if everything is working upon power-on.
2. The system shall stop if there is a problem during startup.
3. The system shall become ready to use if everything is fine during startup.
4. The system shall allow the user to reset it when it is ready to use.
5. The system shall allow the user to activate it when it is ready to use.
6. The system shall allow the user to turn it off when it is ready to use.
7. The system shall start monitoring if it is activated.
8. The system shall start the alarm and send alerts if a sensor is triggered during monitoring.
9. The system shall continue monitoring if it turns out to be a false alarm or the time ends.
10. The system shall stop and return to normal when the user enters the password at any time.
