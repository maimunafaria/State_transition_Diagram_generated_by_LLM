When a user starts interacting with the control panel, they enter a password which the system checks.

If the password is correct, the user is allowed to continue and make selections. If the password is wrong, the user can try again, but only a limited number of times.

If too many incorrect attempts are made, the system becomes locked and does not allow further access for a certain period. After that time passes, it allows the user to try again.

Once access is successfully granted, the system continues normally
