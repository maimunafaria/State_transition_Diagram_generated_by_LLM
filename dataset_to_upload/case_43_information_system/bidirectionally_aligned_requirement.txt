A training request starts when it is created.

After that, it is sent for approval. If it is rejected, the process ends. If it is accepted, the person looks for a suitable session.

Once a session is chosen, enrolment is done. If enrolment is confirmed, the request is satisfied.

From there, the session can be completed, or it can be cancelled. It is also possible to cancel a selected session and go back to choose again.

At any point before completion, the request can be cancelled, which ends the process.
