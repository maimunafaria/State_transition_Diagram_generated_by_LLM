A digital watch lets the user view and adjust the time. By pressing a button, the user can switch between changing the hour, minutes, alarm hour, and alarm minutes. While adjusting, the user can increase the values as needed.

The watch also has an alarm feature. The user can turn the alarm on or off. When the set time matches the current time, the alarm starts ringing. The user can stop it by pressing the alarm button again.

In addition, the watch has a light. When the user presses the light button, the display lights up, and it turns off when the button is released.
