A user enters the platform by registering or signing in. Once access is granted, the user searches for a product, and the system displays the matching results.

After viewing the search results, the user may choose to apply filters to narrow down the options. When filters are applied, the system shows the refined results.

The user can select a product either directly from the search results or after viewing the filtered results. Once a product is selected, the system retrieves the product from the official website and redirects the user there.
