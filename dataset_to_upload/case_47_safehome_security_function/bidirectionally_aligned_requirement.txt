The system starts when the power is turned on and checks if everything is working. If there is a problem, it stops. If everything is fine, it becomes ready to use.

When it is ready to use, the user can reset it, activate it, or turn it off.

If activated, the system starts monitoring. If a sensor is triggered, the alarm starts and alerts are sent.

If it turns out to be a false alarm or the time ends, it continues monitoring.

At any time, the user can enter the password to stop it and bring it back to normal.
