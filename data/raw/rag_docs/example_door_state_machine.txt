Example: Door State Machine

Requirement:
A door can be closed, open, or locked. The default state is Closed.
From Closed, the user may open the door.
From Open, the user may close the door.
From Closed, the user may lock the door.
From Locked, the user may unlock it, returning to Closed.

PlantUML:
@startuml
[*] --> Closed
Closed --> Open : open
Open --> Closed : close
Closed --> Locked : lock
Locked --> Closed : unlock
@enduml
