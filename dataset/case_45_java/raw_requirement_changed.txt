The system reads through a piece of code and skips anything that is not meaningful, such as spaces or comments.

At the beginning, it ignores empty spaces and line breaks. If it finds a line comment (starting with //), it skips everything on that line until it reaches the end.

If it finds a block comment (starting with /*), it continues skipping everything until the comment is closed.

It keeps doing this, moving forward through the text, until it reaches something that is not a space or a comment. At that point, it stops skipping and continues normal processing.
