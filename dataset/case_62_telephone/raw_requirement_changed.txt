When a phone is not being used, it stays idle. As soon as someone picks up the receiver, the system gives a dial tone, letting the user know they can start dialing a number.

If the user begins entering digits, the system starts processing the number. If the number is incomplete or incorrect, the system informs the user or plays a message. If the user stops dialing for a while, a message is also played to prompt them.

Once a valid number is entered, the phone tries to connect the call. If the line is busy, the user hears a busy tone. If the connection goes through, the phone starts ringing on the other end.

When the person being called answers, the call is connected and both parties can talk. If either person hangs up, the call ends and the phone returns to its idle state.

At any point, the call can also be stopped or terminated, ending the interaction.
