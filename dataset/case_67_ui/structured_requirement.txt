Ui — Polished Requirement Specification

Overview
The requirement here is that the users must be able to check whether the budget for an advertising campaign has been exceeded or not. This is calculated by summing the cost of all the adverts in a campaign, adding a percentage for overheads and subtracting the result from the planned budget. A negative value indicates that the budget has been overspent.

Operational Context
Primary actors include user.

Functional Requirements
1. The system shall support describe the user interface behaviour.
