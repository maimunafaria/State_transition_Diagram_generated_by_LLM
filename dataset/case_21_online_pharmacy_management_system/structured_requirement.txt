Online Pharmacy Management System — Polished Requirement Specification

Overview
The customer goes to the shop and purchases the medicine required. So a lot of time is wasted and the person gets tired. If he wants to exchange the product, once again he goes to the shop and replaces them.

Operational Context
Primary actors include admin, customer.

Functional Requirements
1. The system shall allow customers to select required medicines with a single click.
2. The system shall require customers to create a login account prior to placing an order and fill all the details like name, 
address, any id no etc.
3. The system shall enable customers to view the status of medicines.
4. The system shall allow admins to provide the supplier details.
