When a person is dealing with a project, they can either upload a new request or respond to an existing one.

If it is a BPO organization, they enter the project details and save them.

If it is a client, they first search for a suitable project category. Then, the BPO organization can either accept or decline the request. If it is declined, it ends there.

If the BPO accepts, the client then decides whether to proceed or reject it. If the client rejects it, it ends. If accepted, the project moves forward.

After that, the project status is updated. The client may choose to track the progress, while the BPO organization can update the progress.

Once the work is completed, the final project is uploaded. A quality check is done. If the client is not satisfied, the process continues again. If satisfied, payment is made.

After payment, the project is delivered, and the client provides a rating. Then it ends.
