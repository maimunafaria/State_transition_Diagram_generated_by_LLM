Microwave Oven — Polished Requirement Specification

Overview
The microwave starts by showing the current time while it waits for the user. The user can choose the power level, either full power or half power, and can switch between them if needed. After selecting the power, the user sets the cooking time by entering a number.

Operational Context
Primary actors include user.

Functional Requirements
1. The system shall display the current time when it is waiting for user input.
2. The system shall allow the user to choose between full power and half power levels.
3. The system shall enable the user to set cooking time by entering a number.
4. The system shall remain inactive if the door is open during operation.
5. The system shall stop immediately when the door is opened during the cooking process.
6. The system shall allow the user to cancel the cooking process.
7. The system shall return to displaying the current time after stopping or cancelling and wait for the next use.
