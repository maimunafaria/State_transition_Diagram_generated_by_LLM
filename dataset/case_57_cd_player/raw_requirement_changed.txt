A CD player usually starts in a stopped condition, waiting for the user to play a disc. When a disc is inside and the user presses play, the player begins playing the music.

While the music is playing, the user can pause it at any time. When paused, the music stops temporarily, but it can easily continue from the same point when play is pressed again.

The user can also stop the player at any time, whether the music is playing or paused. This returns the player to its stopped condition.

If there are multiple discs, the player can switch to another disc and continue playing. When there are no more discs left, the player simply remains stopped.

This behavior allows the user to easily control playback, pause when needed, switch discs, and stop the music whenever they want.
