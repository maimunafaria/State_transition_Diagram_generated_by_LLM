Weather Monitoring System — Polished Requirement Specification

Overview
This Weather Monitoring Case study allows people to directly check the weather status online. The functionalities are Weather Status, Live Updating, Alert, Graphical Representation, Future Prediction and Weekly. Weather Status – With this function, temperature, humidity, rainfall, pressure and dew point are measured using sensors.

Operational Context
Primary actors include user.

Functional Requirements
1. The system shall measure temperature using sensors.
2. The system shall live update weather data for user view.
3. The system shall provide tips and alerts on certain weather conditions.
4. The system shall plot graphs based on temperature variations for easier analysis.
5. The system shall predict the future weather for the next few days.
6. The system shall display the average, highest, and lowest weekly temperatures.
