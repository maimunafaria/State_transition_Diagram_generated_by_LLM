Oven — Polished Requirement Specification

Overview
The statechart shows the different states for cooking food. The initial state is Door Shut. Consider a scenario that starts when the user opens the door.

Operational Context
Primary actors include user.

Functional Requirements
1. The system shall transition from the Door Shut state to the Door Open state when the user opens the door.
2. The system shall transition from the Door Open state to the Door Open with Item state when the user places an item in the oven.
3. The system shall transition from the Door Open with Item state to the Door Shut with Item state when the user closes the door.
4. The system shall enter the Ready to Cook state after the user inputs the cooking time.
5. The system shall transition from the Ready to Cook state to the Cooking state when the user presses the Start button.
6. The system shall leave the Cooking state and reenter the Door Shut with Item state when the timer expires.
7. The system shall transition from the Cooking state to the Door Open with Item state if the door is opened during cooking.
