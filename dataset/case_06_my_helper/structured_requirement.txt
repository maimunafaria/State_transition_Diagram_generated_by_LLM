My Helper is a mobile application intended to help people in situations where they 
forget their phone at home, lose their phone, keep it on silent, or need to lock/locate
it when it is not nearby. If the application is pre-installed and configured, the user 
can send an SMS to their own phone number from any accessible phone to trigger the required 
help action. The system focuses on providing instant support using SMS-based remote access
without requiring internet connectivity. Key features include remote access without internet,
tracking the phone through SMS, operating without OTP/ID-password, and changing the phone’s 
sound profile (silent to normal) via SMS. The application is intended to be user-friendly 
and usable even on basic smartphones.
