Ott Based System Mini Reel — Polished Requirement Specification

Overview
An over-the-top (OTT) media service is a streaming media service offered directly to viewers via the internet. The OTT media can be streamed anywhere with just the use of an internet connection. The scope of this project lies in the maximised use of the internet by the current generation.

Operational Context
Primary actors include administrator, users.

Functional Requirements
1. The system shall allow users to stream shows and movies anywhere with an internet connection.
2. The system shall allow users to access a trial package of services for 1 month before creating an account.
3. The system shall require payment to be made when creating an account.
4. The system shall allow users to create an account with a secret password for security purposes.
5. The system shall enable premium users to provide suggestions of shows in the discussion room.
6. The system shall provide all users with access to a chat room for chatting and expressing views and comments.
7. The system shall allow the administrator to have control over shows broadcasted and maintain a proper timeline.
8. The system shall provide news feeds highlighting recent updates in MINIREELS in brief format.
9. The system shall allow users to add favourite videos to the LIKES list and watch them whenever needed.
