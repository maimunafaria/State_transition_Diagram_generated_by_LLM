Microwave Oven Operation — Polished Requirement Specification

Overview
One way to do this is by using the notion of a superstate that encapsulates a number of separate states. This superstate looks like a single state on a high-level model but is then expanded to show more detail on a separate diagram. The Operation state includes a number of sub-states.

Operational Context
Primary actors are not explicitly stated in the source text.

Functional Requirements
1. The system shall have a superstate that encapsulates multiple states.
2. The system shall expand the Operation state to show more detail.
3. The system shall perform a status check at the start of operation.
4. The system shall indicate an alarm and disable operation if problems are discovered during operation.
5. The system shall run the microwave generator for a specified time during cooking.
6. The system shall sound a buzzer on completion of cooking.
7. The system shall move to the disabled state if the door is opened during operation.
