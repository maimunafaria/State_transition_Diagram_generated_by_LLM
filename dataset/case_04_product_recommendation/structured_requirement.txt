Product Recommendation System

This case study focuses on using web scraping to compare products across websites, reducing the time and effort involved in finding and comparing products. The system is designed to assist online shoppers by providing an easier way to buy products.

System Functionalities

Registration and Login
The system allows users to register or log in in order to access the system's functionalities.

Product Search
After logging in, the user can search for products in various categories such as medicines, clothes, and machinery.

Search Filter
The system allows the user to specify which website they want to search for the product on, for example, Amazon.

Product Filter
The system lets users filter the product results based on preferences such as rating, reviews, and number of purchases.

Product Purchase
Once the user selects a product, they are redirected to the product's website where they can make the purchase.
