A user can either create an account or continue to the video section directly.

If the user creates an account, they can then move forward to watch videos. From there, they may either go into a discussion or continue to view videos.

The user can also upload videos and then view them.

While viewing videos, the user may either finish there or save videos in their likes list.

If videos are saved in the likes list, the user can view that list and then upload videos based on those liked videos.

If the user enters the discussion option, the discussion takes place and then ends.
