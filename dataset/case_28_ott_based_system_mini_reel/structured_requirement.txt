Ott Based System Mini Reel — Polished Requirement Specification

Functional Requirements
1. The system shall allow a user to create an account.
2. The system shall enable a user to watch videos after they have created an account.
3. The system shall permit a user to upload videos.
4. The system shall allow a user to save videos in their likes list during video viewing.
5. The system shall enable a user to view their saved videos from the likes list.
6. The system shall allow a user to enter the discussion option.
