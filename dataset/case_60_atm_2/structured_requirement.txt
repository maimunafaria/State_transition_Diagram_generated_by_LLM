Atm Scenario 2 — Polished Requirement Specification

Functional Requirements
1. The system shall allow the user to select an amount to withdraw from a list of options.
2. The system shall permit the user to enter a custom value if they want a different amount.
3. The system shall verify the transaction after the user enters an amount.
4. The system shall allow the user to re-enter the amount if the transaction is rejected.
