A user comes to the ATM, and it greets them. Once the card is inserted, the machine begins interacting with the user.

It first asks for the PIN. The user enters it, and the system checks if it is correct. If the PIN is wrong, the user is asked to try again. If the card is invalid or the PIN is entered incorrectly too many times, the ATM keeps the card for security.

At any moment, the user can cancel the operation, and the machine will return the card.

If the PIN is correct, the ATM shows the available options. The user can now choose what they want to do—such as withdrawing money, checking information, or making a transfer.

Once a choice is made, the ATM moves forward to process that request.
