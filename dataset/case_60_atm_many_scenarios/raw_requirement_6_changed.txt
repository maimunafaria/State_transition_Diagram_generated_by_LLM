PIN. The user enters the PIN, and the system checks if it is correct.

If the PIN is wrong, the machine asks again. But if the user keeps entering the wrong PIN, or if the card is invalid (like expired or stolen), the ATM keeps the card for security.

At any moment during this stage, the user can cancel the process, and the machine will return the card.

If the PIN is correct, the user is shown options and chooses to withdraw money. The ATM processes the request and checks whether it can approve the transaction.

If the request is rejected, the machine cancels everything and returns the card.

If it is approved, the ATM gives out the cash, prints a receipt, and returns the card.

However, if the machine runs out of cash during this process, it stops the operation and shows that the system is unavailable.

Finally, after completing or ending the session, the ATM resets itself and becomes ready for the next user.
