A person approaches the ATM and inserts their card. The machine asks for a PIN, and the user enters it. The system checks whether the PIN is correct.

If the PIN is wrong, the user is asked to try again. If this happens too many times, or if the card is invalid, the ATM keeps the card for security reasons.

If the PIN is correct, the machine shows the available options and waits for the user’s choice.

At any point during this process—whether entering the PIN, verifying it, or choosing an option—the user can decide to cancel the operation. When that happens, the ATM simply returns the card.
