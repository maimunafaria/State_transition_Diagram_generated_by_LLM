A person walks up to the ATM, which is waiting for someone to use it. They insert their card, and the machine asks for a PIN.

The user enters a PIN, and the system checks whether it is correct. If the PIN is wrong, the ATM informs the user and asks them to try again.

This process can repeat, but only up to a limit. If the user enters the wrong PIN too many times, or if the card is invalid (like expired or reported stolen), the ATM keeps the card for security reasons.

However, if the PIN is correct, the machine proceeds and shows the available options, allowing the user to choose what they want to do next.
