Digidoclocker — Polished Requirement Specification

Overview
Maintaining the authentic documents physically in a single location is considered to be a challenging task. Thus, digitizing documents enables centralized storage of all the physical documents enabling easy retrieval from anywhere at anytime. It also prevents the troubles caused by missing any of those physical certificates.

Operational Context
Primary actors include user, citizen.

Functional Requirements
1. The system shall allow users to register.
2. The system shall allow authorized users to login.
3. The system shall enable users to upload documents such as driving licence,
vehicle registration, academic mark sheet and other relevant documents 
in various file formats.
4. The system shall display user profile information from the database.
