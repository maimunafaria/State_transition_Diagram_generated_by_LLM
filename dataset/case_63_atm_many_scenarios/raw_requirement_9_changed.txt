After entering the correct PIN, the user is shown different options. The user can choose to transfer money, check information, or withdraw cash.

Once a choice is made, the ATM starts processing that specific request. If the user selects transfer, it processes the transfer. If they choose to check information, it processes the query. If they choose withdrawal, it processes the withdrawal.

If the request is approved, the machine completes the action. For transfers and queries, it prints a receipt. For withdrawals, it dispenses cash.

However, if the request is rejected, the ATM cancels the operation, shows a message, and returns the card.
