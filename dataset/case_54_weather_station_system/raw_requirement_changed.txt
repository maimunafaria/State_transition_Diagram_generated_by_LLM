The weather station begins in a shut down condition. From there, it can either be restarted and begin running, or it can go into a configuration mode when changes or power-saving settings are needed. Once the configuration is finished, it returns to running.

While the system is running, it can do several different things. It may collect data when triggered by the clock. It may prepare a weather summary when a weather report is requested, and after the summary is ready, it sends the information out. It may also check its own status when a status report is requested, and then transmit that result as well. After each of these activities, it goes back to running normally.

The system can also be moved into a controlled mode through remote control. And if a shutdown command is given while it is running, it returns to the shut down condition.
