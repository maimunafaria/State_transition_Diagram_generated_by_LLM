
A user starts by logging in.

If the login details are correct, they can access the system. If not, they see an error and cannot continue.

After logging in, the user can either view products or update their information.

If they choose to view products, they select items and place an order. The selected items are added, and billing details are prepared. The user then chooses a payment method.

If the payment is successful, the order is processed and delivered. If the payment is not successful, it ends there.

If the user chooses to update their information, they enter new details and save them, and then it ends.
