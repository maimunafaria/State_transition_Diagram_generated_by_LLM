

A user starts by logging in and selecting where they want to go. The system shows the journey details, and the user can check them again if needed.

From there, the user can turn on driving mode or continue with other options.

If the user feels hungry, they can choose a place, view the menu, and place an order.

If an accident happens, the system informs the concerned authority and sends a notification.

The system also sends notifications when needed, such as during driving or in case of an accident.

Once the journey is completed, the user can give feedback, and then it ends.
