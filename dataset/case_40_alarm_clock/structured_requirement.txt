Alarm clock Let’s consider a simplified alarm clock: 1. We can set the alarm to “on” or “off”; 2. When the current time becomes that which is set on the alarm, the alarm clock rings continuously; 3.

Operational Context
Primary actors are not explicitly stated in the source text.

Functional Requirements
1. The system shall allow users to set the alarm to either 'on' or 'off'.
2. The system shall ring the alarm continuously when the current time matches the set alarm time.
3. The system shall allow users to stop the alarm from ringing.
