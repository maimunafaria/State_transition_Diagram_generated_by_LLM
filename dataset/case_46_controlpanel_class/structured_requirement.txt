Controlpanel Class — Polished Requirement Specification

Overview
As part of the activation procedure, the ControlPanel object must determine whether any sensors are open. A responsibility named determine-sensor-status() is defined. If sensors are open, ControlPanel must set a status attribute to “not ready.” Sensor information can be acquired from each Sensor object.

Operational Context
Primary actors are not explicitly stated in the source text.

Functional Requirements
1. The system shall determine whether any sensors are open.
2. The system shall set the status attribute to ‘not ready’ if sensors are open.
3. The system shall work in collaboration with Sensor.
